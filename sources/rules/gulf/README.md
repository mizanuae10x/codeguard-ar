# Gulf Security Rules — قواعد أمن الخليج

قواعد إضافية خاصة بمنطقة الخليج والامتثال للأنظمة المحلية.

## الأنظمة المغطاة

| النظام | الجهة | الأولوية |
|--------|-------|----------|
| CBUAE | مصرف الإمارات العربية المتحدة المركزي | P1 |
| DESC | هيئة دبي للأمن السيبراني | P1 |
| UAE IA | الهيئة الوطنية للأمن السيبراني | P1 |
| ADGM | مركز أبوظبي العالمي | P2 |
| DIFC | مركز دبي المالي العالمي | P2 |
| SAMA | البنك المركزي السعودي | P2 |
| NCA | الهيئة الوطنية للأمن السيبراني السعودي | P2 |

## القواعد المخططة

### CBUAE
- `codeguard-0-cbuae-encryption` — تشفير البيانات حسب متطلبات CBUAE
- `codeguard-0-cbuae-access-controls` — ضوابط الوصول للأنظمة المالية
- `codeguard-0-cbuae-incident-response` — الاستجابة للحوادث

### DESC
- `codeguard-0-desc-cloud-security` — أمان السحابة
- `codeguard-0-desc-iot-security` — أمان إنترنت الأشياء
- `codeguard-0-desc-ai-security` — أمان الذكاء الاصطناعي

### UAE IA
- `codeguard-0-uaeia-governance` — الحوكمة
- `codeguard-0-uaeia-risk-management` — إدارة المخاطر
- `codeguard-0-uaeia-asset-management` — إدارة الأصول
